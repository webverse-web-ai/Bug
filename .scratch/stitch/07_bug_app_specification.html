# Bug: AI-native Business Operating System (BOS) ## Product Overview Bug is an AI-native BOS structured around a hierarchy of agents (Marshal, CEO, Agents Hub). It manages everything from order fulfillment (Pulse) to accounting (Tally) and team communication. ## Screen List & Requirements ### 1. Auth / Login Page - Split-screen. - Left: Brand/Welcome message. - Right: Form with Solo/Team toggle, Email/Password, and "Invite Team" input for business flow. ### 2. CEO Dashboard (Index) - Perplexity-style interface. - Central minimalist search/command bar to interact with the Bug agent. - Ability to create new sub-agents. ### 3. Pulse (OMS) - KPI cards (Total Orders, Pending, Revenue). - Data table for order status. - Kanban board for fulfillment. ### 4. Tally (Accounting) - Cash flow line charts and expense bar charts. - Recent invoices and ledger table. ### 5. Team Chat - Slack-style UI. - Channels/DMs sidebar. - Message history and input. - NO floating widget. ### 6. Marshal Chat - Dedicated group chat with the General Manager AI. - NO floating widget. ### 7. Agents Hub - Grid of agents: Pre-built (Tick) and Custom (Ram). - Detail view: 1-on-1 chat interface. ### 8. Settings & Integrations - Profile/Theme tab (accent color customization). - LLM Integrations tab: Google (OAuth), OpenAI (API Key), Anthropic (API Key). ## Global Navigation - Sidebar: Dashboard, Pulse, Tally, Team Chat, Marshal Chat, Agents Hub. - Profile dropdown: Settings, Logout. - Floating Marshal Widget: Bottom-right (except in chat pages).
